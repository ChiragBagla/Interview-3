README

The folder contains 
City: Definite of node on the graph.
Edge: Definite of the route on the graph.
RouteMap: Store all the route on the graph.
GetRoute: Provide various route search methods based on the requirement.
ComputeDistance: Calculator for the distance along a route.
GetShortest: Implementation of a Dijstra Algorithm for the shortest route between two distinct cities.
Trains: Engine for all the answers to the questions.

Test : main method. 

Test/SampleData.txt: Sample data.
Test/input1.txt: another general test data.
Test/input2.txt: null
Test/input3.txt: Invalid route input.

Please run it as:
$javac Test.java
$java Test ./Test/SampleData.txt
